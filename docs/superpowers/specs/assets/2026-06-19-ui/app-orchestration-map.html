Not screens — primitives that appear at all altitudes.

Peek panel
opens from Inbox item · List row · Board card · Graph node → task summary (forward) / delivered (done) → Launch / Open

Shortcuts skip tiers: an Inbox item or ⌘K jumps straight to a Session (0→3); a gate that needs you pushes back up to the Inbox (3→0).

The whole thing, orchestrated

Zoom hierarchy: Attention layer → Project home → Orchestrate → Session — each drills into the next, and ⌘K / Inbox let you teleport across. Cross-cutting primitives (rail, peek, cockpit, feature-shipping, themes) reappear at every altitude instead of being one-off screens — which is why the flows (1–11) and the three concerns stay consistent everywhere.
